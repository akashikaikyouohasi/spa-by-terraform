# AWSのコストについて
## このドキュメントの目的
SPAを作成する際に使用するAWSのサービスを一覧化した上で、それぞれのサービスでどの程度の料金がかかるかを見積もる。

## 使用するサービス一覧
### AWS
- S3: オリジンコンテンツ配置、アクセスログ出力
- CloudFront: CDN
- Route53: カスタムドメイン
- ACM: 証明書発行
- API Gateway: REST API、Auth0認証リクエスト
- Lambda: コンピュートリソース
- WAF: WAF
- Shield: DDos対策
- DynamoDB: データベース
- 

### GitHub
- GitHub Actions: CI/CD

### Others
- Auth0: 認証と認可
